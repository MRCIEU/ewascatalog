To upload data please fill in the details below and upload a results file using the template as a guide. If the results file you upload does not match the format of our template the upload will fail. Click here to be taken to an example of a dataset uploaded to Zenodo via The EWAS Catalog pipeline. If you have any problems filling in the form, please email The EWAS Catalog team at ewascatalog@outlook.com.
